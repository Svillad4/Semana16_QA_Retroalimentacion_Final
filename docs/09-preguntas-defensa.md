# 09 - Preguntas para defensa del proyecto

Prepara respuestas para estas preguntas antes de la exposicion final.

1. Que problema resuelve tu proyecto?
2. Cual es la funcionalidad principal?
3. Que tecnologias usaste y por que?
4. Como comprobaste que JSON Server funciona?
5. Que bug fue mas importante y como lo corregiste?
6. Que evidencia demuestra que aplicaste correcciones?
7. Que retroalimentacion recibiste?
8. Que mejora futura implementarias primero?
9. Como mejoraste la usabilidad o la claridad del proyecto?
10. Que aprendiste del proceso de QA y mejora continua?
