# 08 - Guia de evidencias

Las evidencias son capturas o archivos que demuestran que realizaste pruebas y correcciones reales.

## Evidencias minimas

| Carpeta | Que debes guardar |
|---|---|
| evidencias/antes | Capturas de errores antes de corregirlos |
| evidencias/despues | Capturas despues de aplicar correcciones |
| evidencias/consola | Consola del navegador sin errores criticos |
| evidencias/json-server | JSON Server activo y datos visibles |
| evidencias/responsive | Proyecto probado en vista movil |
| evidencias/coevaluacion | Retroalimentacion recibida o realizada |
| evidencias/plan-mejora | Evidencia del plan de mejora final |

## Consejos

- Usa nombres claros: `bug-001-antes.png`, `bug-001-despues.png`.
- No dejes capturas sueltas fuera de las carpetas.
- Cada evidencia debe coincidir con lo escrito en los documentos.
